# PHP-Music Client Web Player

A lightweight, draggable web-based music player injected directly into your browser. It connects to any PHPMusic API instance to stream your songs, playlists, or artist tracks without leaving the webpage you are currently on.

## Features

- **Two Modes of Operation**: Use the floating/popup Bookmarklet, or run it as a standalone tab using the pure Data URL.
- **Draggable UI**: Move the player anywhere on your screen (when in Floating Window mode).
- **Native HTML5 Audio**: Stable, fast audio streaming natively handled by the browser.
- **Settings Drawer**: Easily input your API URL and Key.
- **Playback Controls**: Features clean Material Design SVG icons for Play, Pause, Next, and Previous.
- **Progress Tracking**: Clickable progress bar and current/total time display fixed to the bottom.
- **Pagination Support**: Infinite scroll automatically fetches more tracks as you scroll down the playlist.
- **Persistent Settings**: Saves your API credentials and preferences locally in your browser so you don't have to re-type them.

## How to Use

You can launch the player using either of the two methods below. **To use them, you can create a bookmark first, or simply copy and paste the code directly into your desktop browser's URL/Address bar!**
