# PHP-Music Client Bookmarklet

A lightweight, draggable web-based music player injected directly into your browser via a JavaScript bookmarklet. It connects to any PHPMusic API instance to stream your songs, playlists, or artist tracks without leaving the webpage you are currently on.

## Features

- **Draggable UI**: Move the player anywhere on your screen.
- **Native HTML5 Audio**: Stable, fast audio streaming natively handled by the browser.
- **Settings Drawer**: Easily input your API URL and Key.
- **Playback Controls**: Features clean Material Design SVG icons for Play, Pause, Next, and Previous.
- **Progress Tracking**: Clickable progress bar and current/total time display.
- **Pagination Support**: Infinite scroll automatically fetches more tracks as you scroll down the playlist.
- **Persistent Settings**: Saves your API credentials and preferences locally in your browser.

## Installation

1. Create a new bookmark in your browser.
2. Name it something like **PHPMusic Player**.
3. Copy the minified code below and paste it into the **URL** (or **Location**) field of the bookmark.
